Set up in 3 steps.

1

Run the installer

Double-click Pluma_0.1.0_x64-setup.exe. If Windows SmartScreen appears, click More info → Run anyway (unsigned build).

2

Add your OpenRouter API key

Press Alt+P to open Pluma. Click the ⚙ settings icon → paste your key from openrouter.ai/keys. Free models available.

3

Press Alt+P anywhere

Pluma lives in your system tray. Summon it from any app with Alt+P, or click the tray icon. That's it — you're running.
